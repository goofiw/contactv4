Here is a list of available commands:
    new  - Create a new contact
    list - List all contacts
    show - Show a contact
    find - Find a contact
    add_num - adds a number to the specificed contact id
    last_name - returns an array of contacts with the specified lastname
    first_name - returns an array of contacts with the specified firstname
    email - returns the contact with the specified email
    update - Begins a REPL for changing a contact item (categories:  firstname, lastname, email)
    add_num - Begins a REPL for adding numbers to specified contact_id
